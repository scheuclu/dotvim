# Vim-configuration

## Installation
```bash
git clone https://github.com/scheuclu/dotvim ~/.vim
ln -s ~/.vim/vimrc ~/.vimrc
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```
Then open the vimrc-file via
```bash
vim ~/.vimrc
```
At first usage, some error messages will probably occur.

Install all Plugins by typing
```vim
:PluginInstall
```

YouCompleteme need to be compiled to fully work. Visit the YouCompleteMe homepage for further instructions.
